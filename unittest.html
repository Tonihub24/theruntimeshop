We have written some code for the website's layout and design, which is what the user sees and interacts with. Now, we need to connect the front-end elements, like buttons and forms, to the backend. On the backend, we use SQL/MariaDB as our database for user logins. When a user clicks the login button, the browser sends this information to the backend. PHP uses the four main database variables: $servername, $username, $password, and $database to connect to MariaDB and check whether the user is authorized to access the site. This process ensures that only valid users can log in and access protected content. 

<?php
function add($a, $b) {
    return $a + $b;
}
echo add(2, 3);
?>
